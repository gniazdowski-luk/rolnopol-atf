# rolnopol-atf

Playwright-based end-to-end testing suite for the **Rolnopol** agricultural management system.

## Prerequisites

- [Node.js](https://nodejs.org/) v18 or later

## Installation

```bash
npm install
npm run install:drivers
```

## Usage

```bash
npm test
```
